# Rename Any Country (EU5 Mod)

Allows renaming any country (including yourself) in Europa Universalis V.

[![Steam Workshop](https://img.shields.io/badge/Steam%20Workshop-Subscribe-blue)](https://steamcommunity.com/sharedfiles/filedetails/?id=3639581385)

## Mod Files

- `in_game/gui/ism_context_menu.gui` - Modifies the condition for the "Rename" option in the context menu.
- `in_game/gui/rename_country_dialog.gui` - Changes how countries are renamed to allow for non-colonial nations.

## Installation

### Steam Workshop (Recommended)
Subscribe on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3619130338).

### Manual Installation
1. Clone or download this repository
2. Copy it to `Documents/Paradox Interactive/Europa Universalis V/mod`
3. Enable as normal in game
